

# 🏛️ AI Chatbot Portal กลาง - ระบบบูรณาการข้อมูลหน่วยงานภาครัฐ

## วัตถุประสงค์
พัฒนา Frontend UI สำหรับระบบ AI Chatbot Portal กลาง ที่รวมศูนย์การสอบถามข้อมูลจาก 4 หน่วยงานภาครัฐ ได้แก่ สำนักงาน อย. กรมสรรพากร กรมการปกครอง และกรมที่ดิน โดยใช้ mock data เป็นฐานสำหรับเชื่อมต่อ Backend ในอนาคต

## ธีมการออกแบบ
- **Clean Light Theme** โทนสีขาว-ฟ้าอ่อน สะอาดตา เป็นทางการแบบราชการ
- ภาษาไทยเป็นหลัก คำศัพท์เทคนิคเป็นอังกฤษได้
- Responsive รองรับ Desktop และ Mobile

---

## หน้าที่ 1: หน้าแชทหลัก (Chat Interface)
หน้าหลักที่ผู้ใช้สนทนากับ AI Agent
- **ช่องพิมพ์คำถาม** พร้อมปุ่มส่ง และคำถามตัวอย่างแนะนำ (เช่น "ขอตรวจสอบทะเบียนยา", "สอบถามเรื่องภาษี")
- **พื้นที่แสดงการสนทนา** แสดง Bubble แชทระหว่างผู้ใช้และ AI
- **แสดงกระบวนการทำงานของ AI Agent แบบ step-by-step** ทุกครั้งที่ตอบ:
  - 🔍 กำลังวิเคราะห์คำถาม...
  - 📋 วางแผนการสืบค้น → เลือกหน่วยงาน: อย., กรมสรรพากร
  - 🔗 กำลังสืบค้นจาก สำนักงาน อย. ...
  - 🔗 กำลังสืบค้นจาก กรมสรรพากร ...
  - ✅ รวบรวมและประเมินผลลัพธ์
  - 📝 สังเคราะห์คำตอบ
- **คำตอบสุดท้าย** แสดงพร้อมอ้างอิงแหล่งที่มา (หน่วยงาน + ลิงก์) อัตโนมัติ
- **โลโก้ 4 หน่วยงาน** แสดงในส่วนแนะนำ

## หน้าที่ 2: Dashboard สถิติการใช้งาน
- จำนวนคำถามทั้งหมด, คำถามวันนี้
- กราฟแสดงหน่วยงานที่ถูกเรียกใช้มากที่สุด (Bar/Pie Chart)
- กราฟแนวโน้มการใช้งานรายวัน/รายสัปดาห์
- หมวดหมู่คำถามยอดนิยม
- สถานะการเชื่อมต่อของแต่ละหน่วยงาน (Online/Offline)

## หน้าที่ 3: หน้าจัดการ Agent / หน่วยงาน
- รายชื่อ 4 หน่วยงานที่เชื่อมต่อ พร้อมรายละเอียด:
  - ชื่อหน่วยงาน, โลโก้
  - ประเภทการเชื่อมต่อ (MCP / API / A2A)
  - สถานะ (Active / Inactive)
  - ขอบเขตข้อมูลที่ให้บริการ
  - จำนวนครั้งที่ถูกเรียกใช้
- ปุ่มดูรายละเอียดเพิ่มเติมของแต่ละหน่วยงาน

## หน้าที่ 4: ประวัติการสนทนา
- รายการประวัติคำถาม-คำตอบย้อนหลัง
- ค้นหาและกรองตามหน่วยงาน/วันที่
- ดูรายละเอียดแต่ละ conversation พร้อม step-by-step ที่ Agent ทำ
- สถานะ: สำเร็จ / ไม่สำเร็จ

## โครงสร้าง Layout
- **Sidebar** ด้านซ้าย: เมนูหลัก (แชท, Dashboard, จัดการหน่วยงาน, ประวัติ)
- **Header** ด้านบน: ชื่อระบบ "AI Chatbot Portal กลาง" + ไอคอนผู้ใช้
- **พื้นที่เนื้อหาหลัก** ตรงกลาง

## ข้อมูล Mock ที่จะเตรียม
- ตัวอย่างคำถาม-คำตอบ 4 หน่วยงาน พร้อม step-by-step ของ Agent
- ข้อมูลสถิติสำหรับ Dashboard (กราฟ, ตัวเลข)
- รายละเอียดหน่วยงานทั้ง 4 แห่ง
- ประวัติการสนทนาตัวอย่าง
